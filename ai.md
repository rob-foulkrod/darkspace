## 🤖 GitHub AI Tools

GitHub has built one of the most comprehensive AI-powered developer
platforms in the industry. Below is a collection of the AI tools
available from GitHub today.

### GitHub Copilot

Your AI pair programmer. GitHub Copilot provides context-aware code
suggestions—from single lines to entire functions—right inside your
editor. It works across VS Code, Visual Studio, JetBrains IDEs,
Neovim, and more, understanding your project context to deliver
relevant completions as you type.

### Copilot Chat

A conversational AI assistant embedded directly in your IDE, on GitHub.com, in GitHub
Mobile, and in the terminal. Ask questions in natural language to get help with
debugging, code explanations, refactoring, and generating unit tests—all without
leaving your workflow.

### Copilot Edits

Perform multi-file code transformations and refactorings from a
single chat prompt. Copilot Edits lets you specify which files to
target, review each proposed change, and accept or reject
modifications with full control over the outcome.

### Copilot Code Review

Get AI-powered code review feedback on your pull requests in seconds. Copilot Code
Review analyzes your changes, flags potential issues, suggests improvements, and
generates PR summaries—available in VS Code and directly on GitHub.com.

### Copilot Workspace

Describe an issue or feature in plain language and Copilot Workspace generates an
implementation plan with proposed code changes across multiple files. It maintains
full codebase context and keeps you in the loop at every step, requiring approval
before applying changes.

### Copilot Cloud Agent

An autonomous coding agent that can analyze your repository, create
branches, solve assigned issues, and open pull requests on your
behalf. Assign it a task and it works through the problem
independently—supporting multiple LLMs including models from
OpenAI, Anthropic, and Google.

### Copilot CLI

AI meets the command line. Copilot CLI answers questions, explains
commands, and helps you scaffold projects using plain English—right
from your terminal.

### GitHub Spark

Build full-stack web applications using natural language prompts. GitHub Spark
transforms your ideas into deployable apps with built-in data storage, authentication,
and CI/CD—no setup required. Perfect for rapid prototyping and validating ideas
quickly, whether you're a developer or not.

### GitHub Models

A structured environment to explore, compare, and evaluate AI
models directly within the GitHub ecosystem. GitHub Models provides
prompt management, live experiments, and APIs for integrating
models into your workflows—with enterprise governance and
reporting built in.

### Copilot Extensions

An ecosystem of third-party integrations that bring external context into Copilot.
Extensions from partners like Sentry, Docker, and others let you extend Copilot's
capabilities to fit your specific toolchain and workflows.

### Multi-Model Support

GitHub Copilot supports a range of leading AI models so you can
pick the best one for the task at hand. Choose from OpenAI GPT,
Anthropic Claude, and Google Gemini families to optimize for
speed, reasoning depth, or cost.
